# Cooker

## Setup (DONE)

```
npx wrangler kv namespace create SIGNALS --preview

```

## Develop (local)

```
# local
npx wrangler dev --live-reload
```

## Develop (remote)

```
# remote
npx wrangler dev --remote
```

## Deploy

```
npx wrangler deploy
```
